git init 这个指令是什么意思呢？就是让git来托管当前这一整个文件。

这里先不解释这个过程了，先按照系统教的一步一步操作

此时此刻，你已经掌握了git的基本用法了。50%的时间你用着两个命令(git add, git commit)可以搞定一切了。

git add 把文件夹里的文件放到暂缓去(staging)

git commit -m “msg” 把所有暂缓去的内容一起放到git的分支上。
